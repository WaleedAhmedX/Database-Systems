# Database Systems Project

This is a simple project I made for my 4th semester database systems subject. It is basically a donor management system where I practiced how to connect PHP with MySQL and build a small web app using HTML, CSS, JavaScript, and PHP.

The project includes things like adding donors, editing donor information, deleting donors, viewing the donor list, and also a simple login and logout system.

I made this project using XAMPP for local hosting and MySQL for the database.

How to run it:
1. Install XAMPP and start Apache and MySQL.
2. Open phpMyAdmin and create the database using the SQL file in the donors folder.
3. Place the project folder in the htdocs folder.
4. Open the project in the browser using localhost.

This project was made for learning purposes, so it may not be perfect, but it helped me understand the basics of database operations and web development.
